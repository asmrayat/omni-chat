# Changelog

All notable changes to OmniChat are documented here.

## 1.18.1 - 2026-06-12

- Fixed split-screen history selection so saved conversations reliably load into the review view.
- Added defensive handling for older or partially captured history entries.

## 1.18.0 - 2026-06-12

- Added split-screen history review mode for reopening previous conversations.
- Added active-session restoration in split view.
- Updated Chrome Web Store metadata.
- Added open-source project files, privacy disclosure, review notes, and validation script.

## 1.17.0 - 2026-06-11

- Added shared local conversation history support.
- Improved split view provider management and saved answer capture.
